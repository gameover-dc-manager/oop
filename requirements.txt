# Installation Instructions:
# 1. Run: npm install
# 2. Set up environment variables in Replit Secrets:
#    - DISCORD_TOKEN: Your Discord bot token
#    - GEMINI_API_KEY: Your Google Gemini API key (optional)
#    - SESSION_SECRET: Random string for session encryption
#    - DISCORD_CLIENT_SECRET: Discord OAuth client secret (optional)
# 3. Configure channels and permissions using /setup commands
# 4. Set up dashboard using /dashboard-setup create command

# Optional External Services:
# - Lavalink server for music (configure in music commands)
# - MongoDB/PostgreSQL for persistent data (currently using JSON files)
# - Redis for caching (optional optimization)
